# Proyecto-1---Sales-Data-for-Economics-Data-Analytics
En dicho proyecto, se trabajó con una base de datos extraída de Kaggle, de una tienda que comercializa diversos productos que contiene información tal como: edad, sexo, país. Todo ello con el objetivo de generar un modelo de predicción sobre el beneficio esperado, para crear una oferta personalizada o una política de precios específica para cada tipo de cliente. El fin de dicho documento, es mejorar los niveles alcanzados de rentabilidad de la empresa en cuestión.
